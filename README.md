# lyssovsky.com

Personal site of **Eugene Lyssovsky** — AI Consciousness Researcher, Investigative
Technology Writer, CIO/CTO.

Static single-page site served via GitHub Pages on the custom domain
[lyssovsky.com](https://lyssovsky.com).

## Stack
- Hand-written `index.html` + `style.css` — no build step.
- Fonts: Playfair Display (display), Source Serif 4 (body), Inter (UI).
- Cinematic noir palette: black `#0F0E0C`, brown `#1A1814`, gold `#D4A547`, ivory `#E8DFD0`.

## Local preview
Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
```

## Deploy
Push to `main`. GitHub Pages serves the root of the branch. The `CNAME` file
binds the custom domain.
